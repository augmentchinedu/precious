# Augment Plus: UI/UX Style Guide

## 1. Introduction

This document outlines the official UI/UX Style Guide for Augment Plus. Its purpose is to ensure a consistent, accessible, and visually appealing user experience across all our web applications and services. All frontend development and design work should adhere to these guidelines.

## 2. Color Palette

Our color palette is designed to be modern, clean, and accessible.

*   **Primary (Brand Color)**
    *   `#4A90E2` (Augment Blue) - Used for primary actions, links, and key branding elements.

*   **Secondary (Accent)**
    *   `#F5A623` (Accent Orange) - Used for secondary actions, highlights, and callouts.

*   **Neutrals (UI Foundation)**
    *   `#121212` (Background Dark) - Primary background color for dark mode interfaces.
    *   `#1E1E1E` (Surface Dark) - Used for cards, modals, and elevated surfaces.
    *   `#FFFFFF` (Text Primary) - Primary text color.
    *   `#B3B3B3` (Text Secondary) - For subheadings, descriptions, and less important text.
    *   `#333333` (Border/Divider) - For borders and dividers.

*   **System Feedback**
    *   `#4CAF50` (Success Green) - For success messages and confirmations.
    *   `#F44336` (Error Red) - For error messages and destructive actions.
    *   `#FFC107` (Warning Yellow) - For warnings and alerts.

## 3. Typography

We will use a single, modern, and highly readable font family for all UI text.

*   **Font Family**: `Inter` (or a fallback `sans-serif` system font).

*   **Headings**:
    *   `H1`: 32px, Bold (700)
    *   `H2`: 24px, Bold (700)
    *   `H3`: 20px, Semi-Bold (600)
    *   `H4`: 18px, Semi-Bold (600)

*   **Body & UI Text**:
    *   `Body (p)`: 16px, Regular (400)
    *   `Small Text / Labels`: 14px, Regular (400)
    *   `Button Text`: 16px, Medium (500)

## 4. Spacing & Layout

We will use a 4px base unit for all margins, padding, and layout spacing to maintain a consistent rhythm.

*   `space-xs`: 4px
*   `space-sm`: 8px
*   `space-md`: 16px
*   `space-lg`: 24px
*   `space-xl`: 32px
*   `space-xxl`: 48px

## 5. Core Components (Initial Styles)

### Button

*   **Primary Button**:
    *   Background: `#4A90E2` (Primary Blue)
    *   Text Color: `#FFFFFF`
    *   Padding: `8px 16px`
    *   Border Radius: `4px`
    *   Hover: Lighter shade of Primary Blue (`#63A4F0`)
    *   Disabled: Background `#333333`, Text `#B3B3B3`

*   **Secondary Button**:
    *   Background: `transparent`
    *   Text Color: `#F5A623` (Accent Orange)
    *   Border: `1px solid #F5A623`
    *   Padding: `8px 16px`
    *   Border Radius: `4px`
    *   Hover: Background `rgba(245, 166, 35, 0.1)`
